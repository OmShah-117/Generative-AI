# **EcoAudit Intelligence | AI Compliance Engine**



* Developed a Retrieval-Augmented Generation (RAG) application to automate the auditing of corporate sustainability reports using Python and LangChain.
* Implemented Llama-3.3-70b via the Groq API, achieving near-instant response times for complex document queries.
* Engineered a high-performance semantic search system using FAISS vector database and HuggingFace embeddings (all-MiniLM-L6-v2) to retrieve relevant data from large PDFs.
* Designed a custom Streamlit dashboard featuring professional Glassmorphism UI and a "Strict Accuracy" mode to eliminate AI hallucinations and ensure 100% evidence-backed answers.
* Optimized data processing using RecursiveCharacterTextSplitter to handle varying document granularities for better context retention.
